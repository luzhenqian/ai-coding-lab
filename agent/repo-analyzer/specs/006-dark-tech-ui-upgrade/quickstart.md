# Quickstart: Dark Tech UI Upgrade

## Prerequisites
- Node.js 22+
- pnpm

## Setup Steps

```bash
# 1. Switch to feature branch
git checkout 006-dark-tech-ui-upgrade

# 2. Install new dependencies
pnpm add next-themes motion class-variance-authority clsx tailwind-merge lucide-react

# 3. Initialize shadcn/ui
npx shadcn@latest init

# 4. Install required shadcn/ui components
npx shadcn@latest add button input card badge tabs scroll-area tooltip separator dropdown-menu

# 5. Start dev server
pnpm dev
```

## Key Files to Modify

| File | Change |
|------|--------|
| `src/app/layout.tsx` | Add ThemeProvider wrapper |
| `src/app/globals.css` | Replace CSS variables with dark tech palette |
| `src/app/page.tsx` | Update layout structure, add animations |
| `src/app/components/*.tsx` | Migrate all components to shadcn/ui |
| `components.json` | shadcn/ui config (auto-generated) |
| `src/lib/utils.ts` | `cn()` utility (auto-generated by shadcn) |
| `src/components/ui/*` | shadcn/ui components (auto-generated) |

## Dev Workflow

1. Make changes to components
2. Check dark mode (default)
3. Toggle to light mode and verify
4. Test streaming chat and workflow
5. Verify animations are smooth
