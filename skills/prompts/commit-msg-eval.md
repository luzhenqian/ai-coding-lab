# commit-msg 触发测试

## ✅ 应该触发（10 条）

1. /commit-msg
2. 帮我写一个 commit message
3. 我改完代码了，帮我生成提交信息
4. 看看 git diff，帮我写个规范的 commit
5. 这次改动应该怎么写 git commit message
6. 帮我按照 conventional commit 规范写提交信息
7. 代码提交一下，帮我写个 message
8. 生成一下这次变更的 commit 描述
9. 帮我看看暂存区的改动，写个提交说明
10. 我要提交代码，帮我组织一下 commit 内容

## ❌ 不应该触发（10 条）

1. 帮我看看 git log 最近的提交记录
2. 我想回滚到上一个版本
3. 帮我创建一个新的 git 分支
4. 这两个分支有冲突，帮我合并一下
5. 帮我写一篇关于 Git 工作流的博客
6. 帮我配置 .gitignore 文件
7. 帮我写一个 GitHub Actions 的 CI 配置
8. 我想给这个 PR 写个描述
9. 帮我看看这个文件的 git blame
10. 帮我把这个仓库的 README 更新一下
