# Step 1: 一句话生成基础页面

## 基础模板

```
帮我做一个个人链接主页，类似 Linktree。要求：
- 顶部显示头像（用首字母代替）和名字
- 下面是一句个人简介
- 然后是一组链接按钮，包括 GitHub、Twitter、YouTube、Bilibili、Email
- 页面整体居中，移动端适配
- 风格简洁现代
```

## 变体示例

### 变体 A: 极简描述（适合测试 AI 理解力）

```
做一个个人链接页面，像 Linktree 那样，有头像、名字、简介和几个社交链接。
```

### 变体 B: 带具体内容

```
做一个个人主页，信息如下：
- 名字：小明
- 简介：前端开发者 / 开源爱好者
- 链接：GitHub（github.com/xiaoming）、掘金、知乎、邮箱 hello@xiaoming.dev
- 风格：简洁，居中布局，适配手机
```

## 使用技巧

- 描述「你想要什么」，而不是「怎么实现」
- 先给出最核心的需求，后续再迭代细节
- 如果第一次生成不满意，可以直接说「重新生成一版，这次要更 XXX」
