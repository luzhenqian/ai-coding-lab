# Step 3: 添加功能

## 基础模板

```
给页面加两个功能：
1. 加一个「复制微信号」按钮，点击后把微信号复制到剪贴板，并显示一个「已复制」的提示
2. 所有外链点击后在新标签页打开
```

## 变体示例

### 变体 A: 加深色/浅色模式切换

```
在页面右上角加一个切换按钮，可以在深色模式和浅色模式之间切换，切换时有平滑过渡动画。
```

### 变体 B: 加访问计数

```
在页面底部显示一个简单的访问次数，用 localStorage 记录（不需要后端）。
```

### 变体 C: 加背景音乐

```
加一个小的音乐播放按钮，点击可以播放/暂停背景音乐，默认静音。
```

## 使用技巧

- 功能需求要具体：不要只说「加个按钮」，要说清楚按钮做什么、放在哪里
- 交互细节很重要：说明点击后的反馈（提示、动画、跳转）
- 一次加一个功能，确认没问题后再加下一个，避免一次改太多导致混乱
