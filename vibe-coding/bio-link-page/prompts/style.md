# Step 2: 用自然语言调整风格

## 基础模板

```
把页面风格改成深色背景 + 毛玻璃效果，整体配色偏紫色调，有一点赛博朋克的感觉。链接按钮悬停时要有微微上浮的动画。
```

## 变体示例

### 变体 A: 日系极简

```
改成白色背景、极简风格，字体用无衬线体，间距大一些，整体感觉干净清爽，像日本设计那种留白很多的风格。
```

### 变体 B: 渐变活泼风

```
背景用从粉色到橙色的渐变，按钮用白色带圆角，整体风格活泼可爱，适合年轻人用。
```

### 变体 C: 暗黑极客风

```
纯黑背景、绿色等宽字体，像终端命令行的感觉。链接前面加上 > 符号，鼠标悬停时文字闪烁。
```

## 使用技巧

- 大胆用感性描述：「像 XX 那种感觉」「更高级一点」「太素了加点颜色」
- AI 对颜色、布局、动画的理解很强，可以直接描述想要的氛围
- 不满意就继续说「再调调，XX 部分不太对」，像跟设计师沟通一样
