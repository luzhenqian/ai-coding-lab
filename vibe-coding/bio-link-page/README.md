# 用 AI 10 分钟做一个个人主页

> 零基础，不写一行代码，用 AI 生成一个好看的个人链接页面，然后部署上线分享给朋友。

<!-- ![最终效果](./assets/screenshot-final.png) -->

---

## 你将学到

- Vibe Coding 的基本思路：用自然语言描述需求，让 AI 写代码
- 如何跟 AI 高效沟通：从粗到细，逐步迭代
- 一个完整的 AI 编程流程：描述 → 生成 → 调整风格 → 添加功能 → 部署

## 最终产物

一个类似 Linktree 的个人链接页面，包含：

- 头像 + 名字 + 个人简介
- 社交媒体链接按钮（GitHub、Twitter、B站等）
- 复制微信号功能
- 可以直接部署上线，分享给任何人

## 前置要求

- 一个浏览器
- 一个 AI 编程工具的账号（任选其一即可）

**不需要**任何编程基础。

## 推荐工具

| 工具 | 说明 | 适合谁 |
|------|------|--------|
| [Lovable](https://lovable.dev) | 在线 AI 建站工具，生成即可预览 | 完全零基础，首选 |
| [v0](https://v0.dev) | Vercel 出品的 AI UI 生成器 | 想要更精细控制的用户 |
| [Bolt](https://bolt.new) | 在线 AI 全栈开发工具 | 想要更多自定义的用户 |
| [Cursor](https://cursor.com) | AI 代码编辑器（需本地安装） | 有一定电脑操作基础的用户 |

> 本教程以 **Lovable** 为主要演示工具。其他工具操作类似，提示词通用。

## 教程步骤

### Step 1: 一句话生成基础页面（3 分钟）

打开你选择的 AI 工具，把下面这段话粘贴进去：

```
帮我做一个个人链接主页，类似 Linktree。要求：
- 顶部显示头像（用首字母代替）和名字
- 下面是一句个人简介
- 然后是一组链接按钮，包括 GitHub、Twitter、YouTube、Bilibili、Email
- 页面整体居中，移动端适配
- 风格简洁现代
```

> 完整提示词模板和变体请看 [prompts/init.md](./prompts/init.md)

**关键心法：描述你想要什么，而不是怎么实现。**

你不需要说「用 flexbox 居中」「用 media query 做响应式」——直接说「页面居中，手机上也能看」就够了。

AI 会帮你生成一个完整的网页。先别急着改，看看整体效果。

---

### Step 2: 用自然语言调整风格（3 分钟）

第一版生成后，大概率不完全符合你的审美。没关系，直接告诉 AI 你想要什么风格：

```
把页面风格改成深色背景 + 毛玻璃效果，整体配色偏紫色调，有一点赛博朋克的感觉。
链接按钮悬停时要有微微上浮的动画。
```

你也可以试试其他风格：

- 「改成白色背景，日式极简风」
- 「活泼一点，用粉色到橙色的渐变背景」
- 「像终端命令行那种黑底绿字的极客风」

> 更多风格变体请看 [prompts/style.md](./prompts/style.md)

**关键心法：像跟设计师沟通一样，大胆用感性描述。**

「更高级一点」「太素了」「像 Apple 官网那种感觉」——这些 AI 都能理解。

---

### Step 3: 添加具体功能（3 分钟）

风格满意之后，我们加一些实用功能：

```
给页面加两个功能：
1. 加一个「复制微信号」按钮，点击后把微信号复制到剪贴板，并显示一个「已复制」的提示
2. 所有外链点击后在新标签页打开
```

> 更多功能扩展示例请看 [prompts/feature.md](./prompts/feature.md)

**关键心法：一次加一个功能，确认没问题再加下一个。**

不要一次塞太多需求，容易让 AI 生成的代码出问题。小步快跑，逐步迭代。

---

### Step 4: 替换成你自己的内容

现在把页面里的占位内容替换成你自己的信息：

```
把页面内容换成我的信息：
- 名字：[你的名字]
- 简介：[你的一句话介绍]
- 链接：[你的实际链接]
- 微信号：[你的微信号]
```

---

### Step 5: 部署上线（2 分钟）

#### 方式 A: 使用 Lovable / Bolt 内置部署

这些工具通常自带一键部署功能，点击「Publish」或「Deploy」按钮即可获得一个公开链接。

#### 方式 B: 部署到 Vercel

1. 把代码上传到 GitHub 仓库
2. 打开 [vercel.com](https://vercel.com)，用 GitHub 账号登录
3. 点击「Import Project」，选择你的仓库
4. 点击「Deploy」

几秒钟后你就会得到一个 `https://xxx.vercel.app` 的链接，可以分享给任何人。

#### 方式 C: 部署到 GitHub Pages

1. 把代码推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选 `/`（根目录）
4. 保存后等待几分钟即可访问

## 常见问题

**Q: 我完全不懂代码，也能做吗？**

完全可以。这就是 Vibe Coding 的核心——你只负责描述需求，AI 负责写代码。你不需要理解生成的代码是什么意思。

**Q: 不同的 AI 工具生成的效果不一样怎么办？**

这是正常的。每个工具对同一段描述的理解不同，生成的视觉效果也不同。如果不满意，继续用自然语言调整就好。提示词模板在不同工具之间是通用的。

**Q: 我想用自己的照片做头像怎么办？**

告诉 AI：「把首字母头像替换成图片，图片地址是 xxx」。如果用在线工具，通常可以直接上传图片。

**Q: 部署后别人看不到怎么回事？**

确认部署平台显示「Ready」或「Published」状态。如果用 GitHub Pages，可能需要等几分钟才能生效。清除浏览器缓存后再试。
