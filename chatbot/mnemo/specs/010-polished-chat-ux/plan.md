# Implementation Plan: Polished Chat UI/UX

**Branch**: `010-polished-chat-ux` | **Date**: 2026-03-16 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/010-polished-chat-ux/spec.md`

## Summary

Enhance the Mnemo chatbot's visual quality by adding shadcn/ui components (Avatar, Tooltip, AlertDialog, Skeleton, Textarea) and chat-specific UX patterns (typing indicator, message animations, empty state suggestions, date-grouped conversations, auto-resizing input). All changes build on the existing shadcn/base-ui + Tailwind v4 stack.

## Technical Context

**Language/Version**: TypeScript 5 / React 19 / Next.js 16.1.6
**Primary Dependencies**: shadcn/ui v4 (base-nova style), @base-ui/react v1.3.0, tw-animate-css v1.4.0, Lucide React v0.577.0
**Storage**: N/A (no new data persistence; existing conversation/message APIs unchanged)
**Testing**: Manual visual testing (animations, responsive behavior, tooltip visibility)
**Target Platform**: Web browser (desktop + mobile responsive)
**Project Type**: Web application (chatbot)
**Performance Goals**: Typing indicator visible within 200ms of sending; animations complete within 300ms; no layout shifts during skeleton→content transitions
**Constraints**: Must maintain Chinese (zh-CN) UI language; must not break existing streaming behavior; touch targets ≥ 44x44px on mobile
**Scale/Scope**: 7 component files modified, 5 new shadcn components installed, 1 new utility function, 1 new CSS animation
