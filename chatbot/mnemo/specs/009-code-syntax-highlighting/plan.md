# Implementation Plan: Code Syntax Highlighting

**Branch**: `009-code-syntax-highlighting` | **Date**: 2026-03-15 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/009-code-syntax-highlighting/spec.md`

## Summary

Enable syntax highlighting for code blocks in AI chat responses by configuring the existing Streamdown component with its `@streamdown/code` Shiki plugin. This adds language-colored code blocks, light/dark theme support, copy-to-clipboard, and inline code styling with minimal code changes.

## Technical Context

**Language/Version**: TypeScript 5 / React 19 / Next.js 16.1.6
**Primary Dependencies**: Streamdown v2.4.0 (installed), `@streamdown/code` (to install), Shiki (bundled in @streamdown/code)
**Storage**: N/A (no data persistence for this feature)
**Testing**: Manual visual testing (code block rendering, theme switching, copy button)
**Target Platform**: Web browser (Next.js App Router, client components)
**Project Type**: Web application (chatbot)
**Performance Goals**: No visible flickering during streaming; highlighting must not block rendering
**Constraints**: Must work with existing Tailwind v4 setup; must respect `.dark` class for theme
**Scale/Scope**: Single component change (`message-bubble.tsx`) + CSS imports
